## PARTE I — VISIÓN ESTRATÉGICA

### Problema de Mercado (Global)

```
HOY: 
├─ Restaurante: usa Mesaio (POS + inventario + contabilidad)
├─ Peluquería: usa Belleza+Citas (agendador + historial cliente)
├─ Tienda retail: usa Square (POS simple)
├─ Clínica: usa MiDoctorow (historiales médicos)
├─ Farmacia: usa sistema FARMACIAS.COM
└─ PROBLEMA: 5 empresas, 5 sistemas, CERO integración

Resultado:
├─ PYME mueve datos Excel ↔ Sistema1 ↔ Sistema2 (manual, lento)
├─ Dueño no ve big picture (datos fragmentados)
├─ Costo total: $500-$1000/mes × 5 sistemas = $2500-$5000
├─ Tiempo admin: 20+ horas/semana en tareas manuales
└─ Decisiones ciegas: "¿Cuál es mi margen real?" (imposible saber)

MAÑANA (Con TITAN):
├─ Restaurante, peluquería, tienda: MISMO sistema (instancia diferente)
├─ 1 login → todo (inventario + caja + clientes + empleados + reportes)
├─ Costo: $99-$299/mes (1 sistema vs 5)
├─ Tiempo admin: 2-3 horas/semana (automática)
└─ Decisiones informed: dashboard único, big picture claro
```
